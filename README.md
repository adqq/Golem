# Golem
*Simple bot discord using JDA and maven (Java)*
#### 
**Golem** It's a simple bot discord with **commands**, **embeds**.

* **Welcome messages:** Welcome messages (*join* and *quit*).
* **Commands:** You can use **!user**, **!serverinfo**, **!help**.
